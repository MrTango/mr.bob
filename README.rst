Bob renders directory structure templates

INSTALL
=======

::

    $ pip install mrbob

USAGE
=====

DEVELOP
=======

::

    $ git clone https://github.com/iElectric/mr.bob.git mrbob
    $ cd mrbob
    $ virtualenv .
    $ source bin/activate
    $ python setup.py develop
    $ easy_install mrbob[test]

RUNNING TESTS
=============

::
    $ nosetests
